## Contributing

Thank you for considering contributing to RealEstate API!

### How to Contribute

1. Fork the repository
2. Create a new branch for your work  
   `git checkout -b feature/your-feature-name`  
   or  
   `git checkout -b bugfix/issue-number`
3. Make your changes
4. Commit with clear messages  
   `git commit -m "Add tenant assignment endpoint"`
5. Push to your fork  
   `git push origin feature/your-feature-name`
6. Open a Pull Request to the `main` branch
7. Fill out the PR template (description, related issue, tests)

### Guidelines

- Open an Issue first for new features or large changes
- Follow existing code style (run ESLint if set up)
- Add/update tests if applicable
- Keep PRs small and focused (one feature/fix per PR)
- Update documentation (README, Swagger) if you change endpoints
- Be kind and respectful in all interactions

Questions? Comment on Issues or PRs — we're happy to help!
